## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/your_username/news_headlines_bot.git
cd news_headlines_bot
```

#### 2. Создание виртуального окружения и установка зависимостей (для ручного запуска)

```bash
python -m venv .venv
source .venv/bin/activate  # Для Linux/MacOS
# или
.\.venv\Scripts\activate  # Для Windows

pip install -e .
```

### 3. Настройка переменных окружения

1. Скопируйте `.env.example` в `.env`:

   ```bash
   cp .env.example .env
   ```

2. Настройте переменные в `.env` файле.

### 4. Запуск с помощью Docker (рекомендуется)

```bash
docker compose up --build
```

### 5. Запуск вручную

1. Установите и запустите PostgreSQL
2. Примените миграции:

   ```bash
   alembic upgrade head
   ```

3. Запустите бота:

   ```bash
   python -m app.main
   ```

## ⚙️ Конфигурация

Все настройки приложения находятся в `.env` файле:

| Переменная             | Описание                           | По умолчанию     |
|------------------------|-----------------------------------|------------------|
| `BOT_TOKEN`            | Токен вашего Telegram бота         | - (обязательно)  |
| `ADMIN_ID`             | ID администратора бота в Telegram  | - (обязательно)  |
| `POSTGRES_HOST`        | Хост PostgreSQL                    | postgres         |
| `POSTGRES_PORT`        | Порт PostgreSQL                    | 5432             |
| `POSTGRES_USER`        | Пользователь PostgreSQL            | postgres         |
| `POSTGRES_PASSWORD`    | Пароль PostgreSQL                  | postgres         |
| `POSTGRES_DB`          | Имя базы данных                    | news             |

## 🗄 Миграции базы данных

Для создания новой миграции:
```bash
alembic revision --autogenerate -m "Описание изменений"
```

Для применения миграций:
```bash
alembic upgrade head
```

## 📋 Команды бота

- `/start` - Начать работу с ботом
- `/help` - Показать справку
- `/sources` - Управление источниками новостей
- `/subscribe` - Подписаться на источник
- `/unsubscribe` - Отписаться от источника
- `/my_subscriptions` - Мои подписки
- `/latest_news` - Последние новости

## 🌐 Деплой

Рекомендуемый способ деплоя - использование Docker Compose на сервере с установленным Docker и Docker Compose.

1. Настройте `.env` файл на сервере
2. Запустите приложение:
   ```bash
   docker compose up -d
   ```

Для обновления:
```bash
docker compose pull
docker compose up -d
```
